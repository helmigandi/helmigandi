Cara Membuat Bootable Flashdisk di Linux

14 November 2025, 07:45

Membuat bootable flashdisk atau flashdrive di Linux dapat dilakukan dengan menggunakan perintah dd. Perlu diperhatikan bahwa penggunaan perintah ini akan menghapus semua data yang ada di flashdisk, jadi pastikan untuk melakukan backup data penting sebelum melanjutkan.

I. Persiapan

  1. Pastikan Anda memiliki file ISO dari sistem operasi yang ingin diinstal.
  2. Colokkan flashdisk ke komputer Anda dan pastikan ukurannya cukup untuk menampung file ISO.

II. Identifikasi Perangkat

Identifikasi perangkat flashdisk Anda menggunakan perintah berikut:

lsblk

Contoh flashdisk yang terpasang di laptop saya yang terpasang di sdd1 mount point, untuk setiap sistem operasi biasanya berbeda. Pastikan Anda mengidentifikasi perangkat yang benar agar tidak terjadi kesalahan: 

NAME MAJ:MIN RM SIZE RO TYPE MOUNTPOINT  
sda    8:0   0 111,8G 0 disk  
├─sda1 8:1   0 549M   0 part  
├─sda2 8:2   0 56,3G  0 part  
└─sda3 8:3   0 55G    0 part /  
sdd    8:16  1 3,7G   0 disk  
└─sdd1 8:17  1 3,7G   0 part /media/helmiz/HELMI
sr0    11:0  1 1024M  0 rom

III. Melepaskan Perangkat dari Sistem Operasi

Sebelum membuat bootable flashdisk, pastikan flashdisk tidak sedang digunakan atau ter-mount. Gunakan perintah berikut untuk melepaskan (unmount) flashdisk: 

sudo umount /dev/sdd1

IV. Membuat Bootable Flashdisk

Gunakan perintah dd untuk membuat bootable flashdisk:

sudo dd if=/path/to/your.iso of=/dev/sdX bs=4M status=progress oflag=sync

Keterangan:

  • if=/path/to/your.iso: Ganti dengan path lengkap ke file ISO yang ingin Anda gunakan.
  • of=/dev/sdX: Ganti sdX dengan perangkat flashdisk yang telah Anda identifikasi sebelumnya (tanpa angka partisi, misalnya sdd).
  • bs=4M: Menentukan ukuran blok sebesar 4 megabyte untuk proses penyalinan.
  • status=progress: Menampilkan progres selama proses penyalinan berlangsung.
  • oflag=sync: Memastikan bahwa data ditulis secara sinkron ke perangkat tujuan.

Proses ini mungkin memakan waktu beberapa menit. Tunggu hingga perintah selesai dan menampilkan ringkasan.

Keluarkan dengan aman setelah selesai, dengan perintah:

sync
sudo eject /dev/sdX

Sekarang flashdisk Anda sudah siap digunakan sebagai media instalasi bootable.
