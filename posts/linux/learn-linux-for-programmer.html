Code Example

Here's an example of Python code with syntax highlighting:

def hello_world():
    """A simple hello world function"""
    message = "Hello, World!"
    print(message)
    return message

# Call the function
result = hello_world()

# Loop example
for i in range(5):
    print(f"Count: {i}")

And here's a JavaScript example:

function greet(name) {
    const message = `Hello, ${name}!`;
    console.log(message);
    return message;
}

// Call the function
const result = greet("World");

// Array manipulation
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map(n => n * 2);
console.log(doubled);

Bash/Shell example:

#!/bin/bash

# Update system
sudo apt update && sudo apt upgrade -y

# Install packages
for package in vim git curl; do
    echo "Installing $package..."
    sudo apt install -y $package
done

echo "Installation complete!"
